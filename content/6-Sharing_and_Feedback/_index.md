## General Evaluation

### 1. Working Environment

The working environment at FCAJ is friendly and professional. Everyone is open and willing to share knowledge and help me whenever I face difficulties — especially when handling issues related to React, Redux, or integrating Frontend with AWS Cognito. Working in a team with shared goals helped me stay motivated throughout the 2-month internship.

### 2. Mentor / Admin Support

My mentor explains issues very clearly, especially when I get stuck on tough bugs like syncing state between Player and History or parsing JWT from Cognito. Instead of giving direct answers, the mentor often suggests an approach and lets me find the solution myself — this helped me learn much more deeply. The admin team also supports well with procedures, documents, and AWS Account–related issues.

### 3. Relevance to Academic Major

The actual work is highly relevant to the Information Technology / Information System major. However, many topics aren't covered in depth at school — such as Serverless Architecture, NoSQL data modeling, advanced React Hooks, Redux Toolkit, Tailwind CSS, and Infrastructure as Code. Self-learning these in a mentored environment and applying them immediately to a real project is a very valuable experience.

### 4. Learning & Skill Development

During the 2-month internship, I trained and developed many new skills:

- **Frontend Skills:** React + Vite, Redux Toolkit, Tailwind CSS, responsive design, complex state handling for the Music Player, drag & drop with playlists, dynamic color extraction.
- **Basic Backend Skills:** AWS Lambda, API Gateway, DynamoDB .
- **AWS Skills:** S3, Cognito, Infrastructure as Code (SST Ion).
- **Working Skills:** team Git workflow, merge conflict resolution, code review, writing documentation.
- **System Thinking:** understanding how Frontend and Backend interact in a serverless architecture.

### 5. Company Culture & Team Spirit

The work culture is positive — everyone respects and supports each other. When deadlines are tight, members pair-program or cross-review code to finish on time. I really felt like part of the team rather than just a passive intern.

### 6. Internship Policies / Benefits

The program provides AWS accounts for hands-on practice, and the internal training sessions and in-depth workshops (such as FCAJ Community Day) are very useful. The flexible schedule helped me balance the internship with my school work.

## Additional Questions

### Most Satisfying Aspect

What I'm most satisfied with is being able to participate in a real project from start to finish — from designing the UI, writing components, connecting APIs, to deploying to the cloud and debugging. By the end of the internship, seeing the product running on the cloud with full features (registration, music playback, playlists with drag & drop, search, multi-genre…) was very rewarding. Additionally, the React + Redux and AWS Serverless knowledge I gained will be a solid foundation for my future work.

### Areas for Improvement

- Could organize more periodic group code-review sessions so interns can learn how to write better code.
- Should provide DynamoDB data modeling documentation early in the program — many people are still unfamiliar with it.
- Could add more training sessions on CI/CD and monitoring — essential knowledge for real-world work.

### Would I recommend this internship?

**Yes.** This program is highly suitable for those who want to pursue Cloud Computing, Frontend, or Full-stack Development. The environment is great, mentors are dedicated, and most importantly, you get to work on a real project rather than just learning theory.

---

## Suggestions & Expectations

### Suggestions

- Organize more hands-on workshops on advanced AWS services such as EventBridge, Step Functions, AWS CDK.
- Have sharing sessions between project teams to learn how other teams solve problems.
- Set up a formal code review process from the start of the program so interns get used to giving and receiving feedback.

### Future Expectations

I'd like to continue working in a Cloud / Full-stack environment to develop further in both Frontend and Backend. I'm particularly interested in topics such as React performance optimization, Server-Side Rendering, Microservices architecture on AWS, and AI/ML integration with Bedrock.

### Other Feedback

Thanks to FCAJ and the mentors for creating such a high-quality internship program. This will be a memorable 2 months in my learning journey.
